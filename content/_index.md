# Security Day {{< year >}}

Der **Security Day** ist eine jährliche IT-Sicherheitsveranstaltung an der [Hochschule Offenburg](https://www.hs-offenburg.de/) - von Studierenden des Studiengangs organisiert, für alle Interessierten offen

Seit 2022 (früher *UNITS Day*) findet das Event jedes Jahr statt. Firmen aus verschiedenen Bereichen der IT-Sicherheit präsentieren Vorträge zu aktuellen Themen: von Angriffstechniken und Pentesting über sichere Softwareentwicklung bis zu regulatorischen Anforderungen

Das Event wird von Studierenden der Hochschule Offenburg organisiert und richtet sich an alle, die sich für Security interessieren - unabhängig vom Studiengang
